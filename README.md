## ⚙️ Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/Fargetik/WEB_Lab_6
   cd WEB_Lab_6
   
2. Запустить контейнеры:
```bash
docker-compose up -d --build
```
3. Открытие главной php страницы:
```http://localhost:8080```

---

## 📂 Содержимое проекта

```docker-compose.yml``` — описание сервиса Nginx

```www/index.php``` — PHP-страница. Показ данных из базы данных

```www/ClientFactory.php``` - Класс для создания базы

```www/ElasticExample.php``` - Подключение к базе Elasticsearch
